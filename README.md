# An-Original-Day
YouTube：https://www.youtube.com/watch?v=a1PFH6P-mPU&t=20s

Genre: First-person Puzzle / Narrative VR / Engine: Unity

This is a first-person VR puzzle-narrative game developed in Unity. It tells the poignant story of a veteran struggling with PTSD on a quest to rediscover their lost identity.

As my debut VR project, it leverages immersive mechanics to reconstruct fragmented war memories through stylized visual metaphors. The project aims to evoke empathy for the veteran community while advocating for peace and anti-war awareness through interactive, emotional storytelling.
